HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • an attribute is defined at the beginning of an element
  • Elements inside of other elements are called child elements
  • a tag shows the contents of an elements
  • Semantic HTML elements describe content and the role it plays on a webpage

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • here are 3 ways to style css: inline, internal and external
  • Selector then declaration which is made up of a property: value
  • the css link must be in the metadata

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: brings you to main branch
  • git pull origin main: pulls the most current main branch
  • git add -A: stages your changes
  • git commit: commits changes
  • -m "xyz": adds message

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Arguements are inside () separated by commas if there are more than 1
  • Arrays require 3 pieces of info separated by ; which include the following:
  • -Declare a variable such as var x = 0
  • -Conditional statement such as x < shapes.length
  • -Iteration statement such as x++ to increase count by 1
